# TSX Stock Analyst Agent

## Purpose
This is an automated stock analysis agent that screens TSX-listed stocks using Warren Buffett's intrinsic value model and recommends the top 5 undervalued investment ideas. Results are emailed daily.

## Tech Stack
- **Python 3.10+** with yfinance, openai (for Perplexity)
- **Perplexity Sonar API** for web-grounded qualitative research
- **yfinance** for financial data (no API key needed)
- **Gmail SMTP** for email delivery (via Python smtplib + App Password)

## Environment Variables (REQUIRED for routine)
The following must be set as environment variables in the routine settings at claude.ai/code/routines:
```
PERPLEXITY_API_KEY=pplx-xxxxx
GMAIL_ADDRESS=your.email@gmail.com
GMAIL_APP_PASSWORD=xxxx-xxxx-xxxx-xxxx
```
To get a Gmail App Password: https://myaccount.google.com/apppasswords
(Requires 2-Step Verification enabled on the Google account)

## How to Run

### Routine Mode (default)
```bash
python -m src.main
```

### Full Mode (standalone, needs Anthropic API key)
```bash
python -m src.main --mode full
```

### Other Options
```bash
python -m src.main --refresh    # Force refresh the TSX ticker cache
python -m src.main --debug      # Enable debug logging
```

## Key Files
- `src/main.py` — Main orchestrator (entry point)
- `src/config.py` — All configuration and parameters
- `src/reporting/email_sender.py` — Gmail SMTP email sender
- `src/reporting/report_generator.py` — Markdown report generator

## Output
- **Routine mode**: `reports/YYYY-MM-DD_candidates.json`
- **Full mode**: `reports/YYYY-MM-DD_tsx_analysis.md`

## 📧 Routine Instructions (for Claude Code)

### CRITICAL: Work in small steps to avoid timeouts

The report MUST be written in multiple small steps. DO NOT try to write the entire report in one response — this will cause a stream timeout. Follow the numbered steps below exactly.

### Step 1: Try the Python pipeline

```bash
python -m src.main
```

If this succeeds, read the output JSON and proceed to Step 3.
If it fails (403 errors, blocked APIs), proceed to Step 2.

### Step 2: Gather data (fallback)

If the Python pipeline fails, identify 5 undervalued TSX stocks using available tools or your knowledge. Focus on Buffett criteria: consistent earnings, high ROE, low debt, wide moat, margin of safety.

Pick 5 stocks and note for each: ticker, company name, sector, approximate price, estimated intrinsic value, margin of safety %, and key reason it's undervalued.

### Step 3: Write report HEADER only

Create the file `reports/{today's date}_tsx_analysis.md` with ONLY the header and summary table. Example:


### Step 9: Append footer and disclaimer

APPEND the methodology and disclaimer:

```markdown
## Methodology
This analysis uses Warren Buffett's Owner Earnings DCF Model:
- Owner Earnings = Net Income + D&A - Maintenance CapEx
- 10% discount rate, 2.5% terminal growth, 10-year projection
- Minimum 30% margin of safety required

## Disclaimer
*This report is for educational and informational purposes only. It does not constitute investment advice. Always conduct your own research and consult a qualified financial advisor before making investment decisions.*
```

If data was from training knowledge (not live), add this above the disclaimer:
```

```

### Step 10: Send the email

Run this Python command to email the report:
```bash
python -c "from src.reporting.email_sender import send_report_email; send_report_email('reports/{today's date}_tsx_analysis.md')"
```

Replace `{today's date}` with the actual date in YYYY-MM-DD format.

## Important Rules
- NEVER write more than one stock analysis per step — this causes timeouts
- ALWAYS append to the report file, never overwrite it
- Reports must include the disclaimer at the bottom
- ALWAYS send the email as the final step
- If using fallback data, ALWAYS disclose the data source
- Keep each stock writeup concise: 2-3 short paragraphs max for "Why We Recommend"
